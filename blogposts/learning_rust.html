Learning rust:

Example 1: Variables and mutability

1. by default variables are immutable in rust.
2. rust allows you to declare variables with the same name.
so instead of doing 


    let mut x = 5;
    x = 6;
you can do 

    let x = 5;
    let x = 6; // compiler uses latest definition
just so you can write: 
        
            let spaces = "   ";
            let spaces = spaces.len();
instead of the cpp version 
        
            string spaces = "    ";
            int spaces_length = spaces.length();
which i guess makes it more like python.
